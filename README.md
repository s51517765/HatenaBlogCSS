# HatenaBlogCSS
My blog CSS.

デザインテーマは「Life」を選択し、タイトル下に「読者になる」ボタンを追加。

「読者になる」ボタンも「ザイン.css」で調整。

「ソースコードエリア」の改行を無効。

## タイトル下

「読者になる」ボタンを追加
```
<div class="subscribe-button">
  <a href="https://blog.hatena.ne.jp/s51517765/s51517765.hatenadiary.jp/subscribe">読者になる</a>
</div>
```
